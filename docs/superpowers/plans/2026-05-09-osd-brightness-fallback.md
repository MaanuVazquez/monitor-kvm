# OSD Brightness Fallback Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a safe OSD remote-control fallback for monitors that reject direct WebOS brightness APIs.

**Architecture:** Add typed remote-button support in the WebOS client by using `getPointerInputSocket` and sending one validated pointer-socket button command. Expose that through a protected HTTP route and reveal a compact Display-tab OSD pad only after known brightness unsupported errors.

**Tech Stack:** Bun, TypeScript, Hono, React, TanStack Query, `ws`, Bun test.

```ts
const REMOTE_BUTTONS = new Set<RemoteButton>([
  "UP",
  "DOWN",
  "LEFT",
  "RIGHT",
  "ENTER",
  "BACK",
  "MENU",
  "EXIT",
```

Add this method inside the returned `client` object:

```ts
async sendRemoteButton(button) {
  if (!REMOTE_BUTTONS.has(button)) {
    throw new Error(`Unsupported remote button: ${button}`);
  }

  const result = (await conn.send(
    "ssap://com.webos.service.networkinput/getPointerInputSocket"
  )) as { socketPath?: string };

  if (!result.socketPath) {
    throw new Error("Pointer input socket not available");
  }

  await new Promise<void>((resolve, reject) => {
    const ws = new WebSocket(result.socketPath!, {
      tls: { rejectUnauthorized: false },
      minVersion: "TLSv1" as any,
      maxVersion: "TLSv1.3" as any,
    } as any);

    const cleanup = () => {
      ws.removeListener("open", onOpen);
      ws.removeListener("error", onError);
      ws.removeListener("close", onClose);
      ws.on("error", () => {});
      if (ws.readyState === WebSocket.OPEN || ws.readyState === WebSocket.CONNECTING) {
        try {
          ws.close();
        } catch {
          // ignore
        }
      }
    };

    const onOpen = () => {
      ws.send(`type:button\nname:${button}\n\n`, (err) => {
        cleanup();
        if (err) reject(err);
        else resolve();
      });
    };

    const onError = (err: Error) => {
      cleanup();
      reject(new Error(`Pointer input socket error: ${err.message}`));
    };

    const onClose = () => {
      cleanup();
      reject(new Error("Pointer input socket closed before button was sent"));
    };

    ws.once("open", onOpen);
    ws.once("error", onError);
    ws.once("close", onClose);
  });
},
```

```ts
display.post("/:host/remote/button", async (c) => {
  const host = c.req.param("host");
  const body = (await c.req.json().catch(() => ({}))) as { button?: unknown };

  if (typeof body.button !== "string" || !REMOTE_BUTTONS.has(body.button)) {
    return c.json({ error: "button must be a supported remote button" }, 400);
  }

  try {
    const client = await devicePool.getClient(host);
    await client.sendRemoteButton(body.button as any);
    return c.json({ button: body.button });
  } catch (err: any) {
    if (err.message?.includes("No credentials found") || err.message?.includes("Call pair()")) {
      return c.json({ error: "Device not paired" }, 404);
    }
    return c.json({ error: err.message ?? "Failed to send remote button" }, 500);
  }
});
```
